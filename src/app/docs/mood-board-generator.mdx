## Overview

The Mood Board Generator curates visual references based on keywords, styles, and project context. Describe what you're looking for and get a collection of relevant images and references organized into a mood board.

## How It Works

1. **Describe your vision** — Enter keywords, styles, moods, or reference points.
2. **Add context** — Optionally link to an existing project for contextual awareness.
3. **Generate** — The AI curates a collection of visual references.
4. **Save** — Add generated references to a collection in the References board.

## Example Prompts

- "Futuristic stage design with neon lighting and fog effects"
- "Brutalist architecture meets immersive tech installation"
- "Desert festival VIP lounge, luxury minimalism"
- "LED panel arrays for a 360-degree concert experience"

## Integration

Generated mood boards sync directly with the References board. Each reference is automatically tagged and can be assigned to a project collection.

## Inputs

| Parameter | Type | Description |
|-----------|------|-------------|
| `keywords` | `string[]` | Keywords describing the visual direction |
| `style` | `string` | Visual style (3D, minimal, retro, etc.) |
| `count` | `number` | Number of references to generate (1-20) |
